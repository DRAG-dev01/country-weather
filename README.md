\# Country Weather



A desktop app that searches for country info and automatically fetches live weather for its capital city.



\## Features

\- Search a country and get its capital and population

\- Automatically fetches weather for the capital city

\- City validation using fuzzy matching

\- Error handling for connection issues



\## How to run

1\. Install dependencies: `pip install requests beautifulsoup4`

2\. Run: `python weather_country_app.py`



\## Built with

\- Python

\- tkinter

\- requests

\- BeautifulSoup

\- difflib
